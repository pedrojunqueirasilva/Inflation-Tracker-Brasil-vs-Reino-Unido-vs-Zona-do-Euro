# LinkedIn Post Draft

I have just completed an end-to-end macro analytics project comparing **Brazil, the United Kingdom, and the Euro Area** across inflation, real interest rates, unemployment, GDP growth, and exchange-rate dynamics.

The project answers one central question:
**How did these economies behave through recent cycles, and what does that imply for inflation pressure, monetary policy, and cost-of-living conditions?**

### What I built
- Python data pipeline using public official data APIs (World Bank)
- Clean country-year analytical panel with harmonised indicators
- Insight engine highlighting cycle differences and policy-relevant signals
- Dashboard specification for Power BI/Tableau (+ Streamlit option)

### Why this matters
This work combines business interpretation with practical analytics and shows how data can support strategic and policy thinking in international contexts.

If you work in analytics, strategy, research, or consulting, I would be happy to exchange ideas.
